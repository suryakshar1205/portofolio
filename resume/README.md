# Resume Folder

Upload the latest resume here as a PDF or DOCX file.

The website fetches files from the GitHub repository `/resume` folder at runtime, compares their latest commit dates, and automatically opens/downloads the newest supported file. No code change is required when a newer resume is added.

Supported extensions:

- `.pdf`
- `.docx`

Recommended naming:

- `Suryakshar_Konchada_Resume_2026.pdf`
- `Suryakshar_Konchada_Resume_2026.docx`
